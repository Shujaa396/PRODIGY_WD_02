# Digital Stopwatch ⏱️

A simple and functional digital stopwatch built using [![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/HTML), [![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/CSS), and [![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript). The stopwatch tracks time in **minutes**, **seconds**, and **milliseconds** and includes **start**, **stop**, and **reset** functionalities.

## Features
- **Start Button**: Begin counting the time.
- **Stop Button**: Pause the time counting.
- **Reset Button**: Reset the stopwatch to zero.
- **Real-Time Update**: Time updates dynamically in minutes, seconds, and milliseconds.

## How to Use
1. Click the "Start" button to begin the stopwatch.
2. Click "Stop" to pause the time.
3. Click "Reset" to set the stopwatch back to 0.
